一 写在前面：

  1. Rhinoceros本系列教程的制备，起因为给力手绘所托，进行网络授课。在备课过程中因为事情比较多只能推掉。但最终还是以文字版形式陆续完成了这篇教程，作为我自己学习软件的总结，也希望能帮助到一些想要学习Rhino的未入门初学者。
  2. 我是建筑淆淆士小梁，未毕业，未工作，是个菜鸟。
  3. 第一次讲课。从学生到讲师，诚惶诚恐。可能时长不能把握的很好。内容不能让大家满意。也可能出现纰漏。不周处请谅解。
  4. 短短的几天课时，甚至几周，不能把世界上任何一个软件完全学会。希望大家认可Rhinoceros，认可Grasshopper，不要畏惧，不要抵触，温故知新，熟能生巧。多借助搜索引擎的力量，借助学习前人的经验。软件不会成为设计的瓶颈和绊脚石。
  5. 我不是老师，也不是学长，是和你们一起学习软件的伙伴，只不过比大家早学了一些时间，希望能一起探索，一起进步。

二 正文

1前期准备

1.1Rhinoceros下载

官网https://www.rhino3d.com/download下载

  1. Rhino 6 for Windows - 试用版 - 在90天内试用此全功能版本，90天后将无法保存且插件也无法打开，除非购买一个许可证。
  2. Rhino 6 for Windows - 最新版 - 需要使用 Rhino 6 授权码。

国内授权码购买：淘宝 购买教育版

购买授权码之后会在邮箱收到授权信息，请妥善保管，不要泄露。

1.2Rhinoceros安装

注意选择中英双语。尽量安装在C\Program File\

尽量使用英文界面。

1.3插件安装

平台https://www.food4rhino.com/。

推荐插件：

  1. Sectiontools For Rhino
  2. Sunflower
  3. VisualARQ
  4. TRACE

2界面

默认界面：

  1. 菜单栏
  2. 指令历史窗
  3. 工具集合栏
  4. 常用指令边栏
  5. 工作视窗
  6. 状态栏
  7. 捕捉选项栏
  8. 信息栏

2.1菜单栏

菜单栏的命令包含常用的指令。配合输入指令，图标使用。

Options→Appearance→Show the following items→Menu

2.2 指令历史窗

界面中不可或缺的部分。F2

Options→Appearance→Show the following items→Command prompt

2.3工具列集合栏

可以根据需求增添指令，标签。齿轮可选择锁定。

Options→Toolbars→Standard Toolbar Group

2.4常用指令边栏

滚动鼠标中键，与工具栏关联

Options→Toolbars→Size and Styles→Visibility→Sidebar visible

2.5工作视窗

标签栏

4View ZEA

NewViewport

CloseViewport

2.6面板

F3→Properties

2.7捕捉选项栏

Osnap

2.8状态栏

单位，图层，各种选项

  • 锁定格点
  • 正交
  • 平面
  • 捕捉
  • 智慧轨迹
  • 操作轴
  • 记录建构历史
  • 过滤器

3视窗操作

3.1移动旋转缩放放大最大化最小化视窗

旋转：RMB

平移：Shift+RMB

缩放：Ctrl+RMB

视口摆动：Ctrl+Alt+RMB，Shift+Alt+RMB

切换视窗大小：Double LMC左上角标签

3.2 加选减选框选

点选：LMC

加选：Shift+LMC

减选：Ctrl+LMC

非贪婪框选：左上角LMB

贪婪框选：右下角LMB

3.3切换视窗

  • Ctrl+Tab
  • Status bar LMC

4自定义工具列

4.1默认工具列

Options→Toolbars→Files→Default

C:\Users\==yourname==\AppData\Roaming\McNeel\Rhinoceros\6.0\UI\default.rui

4.2新增工具列

RBC→NEW

4.3导入工具列

Options→Toolbars→Files→Default→Close→Import Toolbars

4.4弹出工具列

Options→Mouse→Middle mouse button→Popup this Toolbar

4.5自定义弹出

移动：Shift+LMB Drag

复制：Ctrl+LMB Drag

5显示模式

5.1更改显示模式

视窗左上角标签栏下拉列表，或输入指令：SetDisplayMode_Mode_

  1. 线框模式 Wireframe
  2. 着色模式 Shaded
  3. 渲染模式 Rendered
  4. 半透明模式 Ghosted
  5. 极地模式 Arctic
  6. 光线追踪 Raytraced
  7. ……

可单独设置物件显示模式 输入指令：SetObjectDisplayMode

5.2增删显示模式

Options→View→Display Mode

New，Copy，Delete，Import ，Export

……

6面板

  • Display
  • Properties
  • Layers
  • Layer States
  • Materials
  • Named CPlanes
  • Named Views
  • ……

6.1 Display

不同显示模式不同选项

  • 切换模式
  • 更改背景颜色
  • 显示选项
  • Backface Color 更改艳丽颜色方便观察
  • Clipping Plane 勾选Show Fills Show Edges

6.2Properties

不同数据类型不同选项

  • 更改名称
  • 更改显示颜色打印颜色
  • 更改线性
  • 更改显示线宽
  • 更改材质
  • 更改UV
  • ……

切换为打印模式命令 PrintDisplay_Color_Print

6.3 Layers

  • Filters
  • Tools
  • 增加图层与子图层
  • 复制图层与复制物件
  • 移动物件所属图层
  • ……

7物件的快速变动

移动：LMB Drag

复制：Alt +LMB Drag

垂直：Ctrl+LMB Drag

正交：F8 或 状态栏Ortho

Gumball：

  • R→X axis ；G→Y axis ；B→Z axis
  • 移动：拖动箭头（LMC设置具体数值）
  • 旋转：拖动弧线（LMC设置具体数值）
  • 缩放：拖动方块（LMC设置具体数值）
  • 白色圆圈进行设置

锁定方向：Tab

三 作业

  1. 定义自己的窗口
  2. 预习指令，思考建模需要那些常用命令。
  3. 思考自己以往的课程设计的建模逻辑，思考使用su建模是否还有优化的可能，如何优化。
