1.前期准备

前期准备基本安装软件与注册账户。

注意： 标题前有 ‘^’ 为非必要项目。

1.1安装Node.js

Hexo是由Node.js编写的静态博客框架，所以需要安装Node.js。官网下载最新LTS版本，默认选项安装即可。

安装完成后使用快捷键 win+R，输入cmd打开命令行，输入

node -v
npm -v

检查是否安装成功。

1.2^安装Notepad++

“自带notepad（自带记事本）太过弱智！”

​ ——廖雪峰。

如果没有其他编辑器，建议安装Notepad++编辑器。默认选项安装即可。

老油条请忽略。

1.3安装Git

Git是分布式版本控制器。建议全世界的搬砖师傅使用。

1.31安装Git

官网下载最新Git，默认安装，过程中注意，选择使用默认命令行，编辑器默认为Vim，也可选择NotePad++等。

1.32^安装TortoiseGit

TortoiseGit是Git的图形化操作工具。平时部署博客，使用命令行即可。如果使用Git推送其他文件或工程，可使用TortoiseGit。命令行用户可忽略。

官网下载最新TortoiseGit，默认安装即可。

1.4新建本地博客文件夹

在本地选择一个你存放博客源文件的地址新建一个文件夹，随意命名。

这个文件夹下的目录未来将进行git操作。

1.5注册GitHub账户

GitHub是全球最大同性交友网站，女装大佬交友网站开源社区。

1.51注册GitHub

官网注册GitHub账户，记住用户名和邮箱。

1.52新建一个Repository

新建一个Repostory，命名为 “yourname.github.io” ，Public或Private视个人情况，Readme文件视个人情况。

注意：“yourname”在引号内替换为GitHub的用户名。

1.53配置Git

ctrl+R输入cmd，进入命令行。

git config --global user.name "yourname"
git config --global user.email "youremail"

注意：“yourname”和”youremail”在引号内分别替换为GitHub的用户名和邮箱。

命令行输入

git config user.name
git config user.email

检查是否正确。

1.54创建密钥

命令行输入

ssh-keygen -t rsa -C "youremail"

注意：“youremail”在引号内替换为GitHub的邮箱。

点击三次回车生成密钥。在电脑C盘Users目录选择使用的用户点击进入，找到文件夹“.ssh”，使用Notepad++打开文件“id_rsa.pub”，复制全部内容。

  1. 进入GitHub，右上角个人头像下拉列表，点击settings。
  2. 点击左侧目录中 ”SSH and GPG keys“ 。
  3. 点击绿色按钮 New SSH key
  4. 随意命名，并在下方输入框中，将刚刚复制的“id_rsa.pub”中的公钥内容，粘贴进去。
  5. 确定。

1.6^安装Typora

Typora为非开源markdown编辑器。简洁大方，推荐使用。老markdown用户请忽略。

2.搭建Hexo

2.1安装Hexo

选中创建的博客文件夹，所以命名，右键Git Bash Here，并在弹出的窗口中输入

npm install -g hexo-cli

输入

hexo -v

检查是否成功。

继续在命令行中输入

hexo init blogname

进行初始化。

注意：blogname替换为文件名

继续在命令行中输入

npm install

查看文件夹内是否生成出文件。

继续在命令行中输入

hexo g
hexo s

在浏览器输入 localhost:4000，将生成默认的博客主页。

2.2将Hexo部署到GitHub

在博客文件夹中，找到文件 “_config.yml”，翻至最后，在

deploy:
  type: git
  repo: https://github.com/YourgithubName/YourgithubName.github.io.git
  branch: master

中，将YourgithubName修改为GitHub账户

在命令行中输入

npm install hexo-deployer-git --save

继续输入

hexo g
hexo d

注意：deploy时可能要你输入username和password。

在浏览器中输入 http://yourname.github.io，打开博客。

3.常用hexo命令

hexo s
hexo g
hexo d
hexo g -d
hexo n "title"

详细参考：https://hexo.io/zh-cn/docs/commands.html
