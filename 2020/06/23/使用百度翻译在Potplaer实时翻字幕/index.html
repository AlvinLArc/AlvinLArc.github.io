  1. 请无梯读者，纯中文字符阅读读者和看视频为了娱乐（看电视剧电影综艺节目等一系列奶头乐）的人关闭本篇教程。本教程只服务热爱学习，但是外语不咋地的孩纸萌。
  2. 本教程是个人学习笔记，借鉴了许多其他教程内容，并非手把手，并非复制粘贴。同时也因能力有限，不能保证所有内容正确且适用于所有人。因阅读后，操作出现问题与损失，后果请由读者自身承担。
  3. 教程针对Windows系统，其它系统不做解释
  4. 感谢支持，笔芯♥，么么哒

1前期准备

前期准备基本安装软件与注册账户。

注意：要把大象装冰箱，拢共分三步。

1.1开通百度翻译

  1. 登录百度账号->上菜单“产品服务” -> 左菜单“通用翻译” -> 点击“立即使用” -> 完成注册（不要填写IP）
  2. 上菜单“管理控制台” -> 左菜单“总览” -> 申请信息中的“APP ID”和“密钥” -> 记住它（一字不落的背下来）

1.2PotPlayer

官网下载最新PotPlayer，安装至C\Program Files\。

1.3SubtitleTranslate

将下载好的SubtitleTranslate - baidu.as、SubtitleTranslate - baidu.ico这两个文件复制到\PotPlayer\Extention\Subtitle\Translate

2配置PotPlayer

打开PotPlayer -> 右键视频窗口 -> 字幕 -> 实时字幕翻译 -> 实时字幕翻译设置 -> 选择“Bai Du translate” -> 账户设置中将背下来的“APP ID”和“密钥”填入 -> 确定

注意：每月200万字限制，以及要将字幕缓冲量加大，否则翻译慢影响观看视频。
